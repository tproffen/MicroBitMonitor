MicroBit Monitor Documentation

Prerequisites

The MicroBit monitor allows to display information and interact with a MicroBit connected via Bluetooth LE. Please note, that not all browsers and devices have web Bluetooth capabilities. The easiest way to get started is to work from one of these examples

  • Accelerometer
  • Exercise monitor
  • Temperature plotter

Configuration

The MicroBit communicates with the web browser via BLE UART commands. The commands are strings in general formed as command:parameter:.. Note that the maximum string length that can be submitted it 20 characters requiring short commands. Here is a list of currently implemented commands.

CommandDescription
config:row1:row2:row3 This sets the configuration of the screen. This needs to be the first command right after the browser send the string 'ready'. Rows 2 and 3 are optional. Each parameter row can take one of the values below representing different panels to be displayed.
Panel commands
t1Created a full width text panel. The command tA:text will change the text on the web page to text
v1Created a full width panel to display a label and value. The command lA:label will change the label on the web page to label. Similarly A:value will change the value to value
v2Created a two panel across a row to display a label and value. The command lA:label will change the left label and lB:label on the web page to label. Similarly A:value will change the value to value on the left side and B:value will do the same for the right panel.
bctlCreated a full width panel with a Start and Stop button. Each button will send its value (e.g. Start) to the MicroBit when pressed.
brstCreated a full width panel with a Reset button.
Chart commands
cbThis will create a full width bar chart using Google charts. The command cL:l1:l2:.. sets the labels for the data columns to l1, l2 and so on. This needs to be the first command after the configuration as it sets up the data dimensions. The data itself is sent using cV:v1:v2,... The number of values needs to match the number of labels. In case of the bar chart, only the last values are shown. The command ct:title allows to set the chart title and cMax:value and cMin:value are setting the range of the chart.
ctThis will create a full width line chart using Google charts. the commands are the same as for the bar chart. The only difference is that data are added to the line graph until the buffer size is reach and future data will scroll across the chart.
